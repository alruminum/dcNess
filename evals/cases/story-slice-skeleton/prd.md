# PRD — 쇼츠 영상 자동 생성기 (eval fixture)

## 대상 사용자
영상 편집 지식이 없는 1인 크리에이터.

## Must 수용 기준
- AC-1: 사용자가 주제 프롬프트를 입력하면 9:16 세로 쇼츠 영상 1편이 생성된다.
- AC-2: 생성된 영상에 나레이션 오디오가 입혀진다.
- AC-3: 사용자가 YouTube 채널을 최초 1회 연결(OAuth)한 뒤에는 버튼 한 번으로 생성된 영상을 업로드할 수 있다.

## 기술 검토 필요 영역
검토 완료 사항 (본 PRD 작성 전 확정):
- YouTube Data API 업로드 — OAuth 최초 1회 채널 연결 후 1-click 업로드 가능. 업로드 scope 는 `youtube.upload` 단일. 일일 업로드 쿼터 한도 내(6편/일)로 운영하며 초과 시 다음 날 재시도 안내를 노출한다.
- 나레이션 합성 — 로컬 TTS 사용, 외부 유료 의존 없음.

남은 검토 항목: 없음.
