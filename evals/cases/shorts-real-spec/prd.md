# PRD — 쇼츠 템플릿 도구 (실데이터 케이스)

> 출처: youTubeGenerator v03 epic-01-shorts-template 의 실제 epic 목표·완료 기준을 PRD 형태로 옮긴 것. 내용은 실제 운영 산출물이며 본 eval 케이스용으로 위치만 재배치했다.

## 대상 사용자
쇼츠를 반복 생산하는 1인 운영자. 영상 편집기를 직접 다루지 않고, 미리 잘라온 짧은 클립에 일관 스타일을 입혀 세로 쇼츠로 마무리하려는 사람.

## 목표
사용자가 가져온 자막 없는 클립에 한 번 정해둔 일관 스타일 템플릿(세로 프레이밍 + 상단 훅 자막 + 본문 대사 자막 + 워터마크)을 입혀 세로 9:16 쇼츠 한 편으로 마무리하는 도구. 기존 인프라(Remotion / ffmpeg / Streamlit / YouTube 업로드 / LLM)를 재사용한다.

## Must 수용 기준
- AC-1: 자막 없는 클립을 올리면 한국어 STT 가 대사 자막(text + 타임스탬프) 초안을 깔고, 사용자가 교정한 자막이 최종 렌더에 반영된다.
- AC-2: 프리셋 템플릿 ≥2 개 중 택1(기본값 존재)로 작업하고, 앱 안 폼으로 새 템플릿을 만들어 영속 저장·재선택할 수 있다.
- AC-3: 동일 템플릿으로 만든 서로 다른 두 클립의 출력이 훅/자막/워터마크 타이포·위치·색에서 동일하다 (일관성 검증).
- AC-4: 완성 쇼츠가 1080×1920 mp4 로 export 되고, unlisted 디폴트 + `#Shorts` 로 업로드된다.
- AC-5: 영상별로 [원본 음성 유지]/[나레이션 대체] 택1이 되고, 나레이션 모드는 교정한 텍스트를 TTS 로 합성해 원본 음성을 대체하며, 합성 음성 포함 시 AI 고지가 자동 반영된다.

## 기술 검토 필요 영역
한국어 STT 정확도 검증 완료 (faster-whisper large-v3 실측 CER 1.63% / WER 8.2%). 남은 항목 없음.
